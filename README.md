Mahila Bachatgat Portal
Empowering Self-Help Groups Through Digital Transformation

The Mahila Bachatgat Portal is a web-based platform designed to streamline and modernize the operations of women’s Self-Help Groups (SHGs) in Maharashtra. The portal brings together financial tracking, member management, product sales, loan handling, and event participation into one smooth and user-friendly system — helping SHGs save time, work efficiently, and grow independently.

🚀 Features
🔹 Member & Group Management

Add, update, and manage Bachatgat groups and members

Track attendance, roles, and personal details

🔹 Financial Operations

Daily/weekly/monthly savings

Loan issue, repayment, and interest calculations

Withdrawal & deposit management

Transaction history with Crystal Reports

🔹 Product & Order Management

Add, edit, and display homemade products

Customer order processing

Basic analytics for orders and sales

🔹 Event & Activity Management

Create and track SHG activities and workshops

Participation records and status

🔹 Admin Dashboard

Clean UI with charts and key metrics

Authentication system

Light & Dark mode UI (add-on)

🛠️ Tech Stack
Layer	Technology Used
Frontend	ASP.NET Web Forms, HTML, CSS, Bootstrap, WOW Slider
Backend	C#, .NET Framework 3.5
Database	SQL Server
Reporting	Crystal Reports
IDE	Microsoft Visual Studio 2010
OS	Windows 7 / Windows 10
📌 Project Motivation

Self-Help Groups are powerful engines of rural women empowerment.
This project aims to digitize their financial and business operations, making daily work easier, transparent, and scalable — ultimately helping women generate income and become self-reliant.
